== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/elematic-addons-for-elementor/` or install via Plugins > Add New
2. Activate the plugin through the "Plugins" screen in WordPress
3. Open any Elementor page and start using "Elematic" widgets in the panel

== Frequently Asked Questions ==

= Does it work with Elementor Free? =
Yes. All widgets are compatible with Elementor Free and do not require Pro.

= Will it slow down my website? =
No. Elematic is optimized for performance and includes only essential assets.

= Can I use it with other Elementor addons? =
Absolutely! Elematic is compatible with all major Elementor addon packs.

= Does it support Lottie animations? =
Yes. A dedicated Lottie widget is included.

= Does your post widget support Load More button to load new post without reaload page? =
Yes. A Post Grid widget is included.

= What makes Elematic different from other Elementor addons? =
Elematic focuses on Webflow and Framer-style animations with performance optimization. Unlike heavy addon packs, it loads only the assets you need, ensuring fast page speeds while giving you 30+ modern, animated widgets.

= How do I create Webflow-style animations in Elementor? =
Simply install Elematic and use widgets like Circle Text, Marquee, Highlight Text, Image Slide, and Service List. These widgets are specifically designed to replicate popular Webflow animation effects directly in Elementor's drag-and-drop builder.

= Is Elematic compatible with popular WordPress themes? =
Yes. Elematic works with any WordPress theme that supports Elementor, including Hello Elementor, Astra, GeneratePress, OceanWP, Kadence, Blocksy, and all other major themes. It's fully tested for compatibility and visual consistency.
