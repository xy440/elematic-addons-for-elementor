== Description ==

🚀 **NEW: Elementor Theme Builder (Header & Footer)**  
Elematic now includes a **built-in Elementor Theme Builder**, allowing you to create **custom headers and footers visually using Elementor**, without editing theme files or writing code.

You can:
- Design **global or conditional headers and footers**
- Edit header, content, and footer layouts seamlessly

This makes Elematic a **complete layout solution**, not just a widget pack.

---

**Elematic** is a fast, lightweight addon for the Elementor page builder, designed for modern WordPress websites. It includes beautifully animated widgets inspired by **Webflow and Framer**, giving you motion-design freedom directly inside Elementor.

Whether you’re building a landing page, business website, portfolio, SaaS layout, blog, or WooCommerce store, Elematic gives you everything you need to design **clean, interactive, and high-converting layouts**.

The plugin is built with **performance first** in mind. Assets load only when widgets are used, ensuring fast page loads, strong Core Web Vitals, and excellent Google PageSpeed scores.

== Elementor Theme Builder (Header & Footer) ==

Elematic includes a **visual Theme Builder** that lets you design headers and footers using Elementor.

**Features:**
- Create unlimited headers and footers
- Assign display conditions (global, pages, posts, archives)
- Edit headers/footers directly from the frontend
- Compatible with classic and block themes
- No theme file overrides
- Safe, lightweight, and cache-friendly

This brings **Elementor Pro-style header/footer editing** into a lightweight, developer-friendly plugin.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/elematic-addons-for-elementor/`
2. Activate the plugin from WordPress admin
3. Edit any page with Elementor and find **Elematic** widgets
4. Create headers and footers from Elematic Theme Builder

== Frequently Asked Questions ==

= Does Elematic include a header and footer builder? =
Yes. Elematic includes a visual Elementor Theme Builder for creating custom headers and footers.

= Does it work with Elementor Free? =
Yes. All widgets and the Theme Builder work with Elementor Free.

= Will it slow down my website? =
No. Elematic is built with performance-first architecture.

= Is the Theme Builder's custom Header, Footer compatible with block themes? =
Currently not, but we are working for it.
